## Maximum Character

Diberikan satu `string`, kembalikan karakter yang jumlahnya paling banyak di string tersebut. Contoh:

- `maximumCharacter("balonku ada lima")` -> `a`

- `maximumCharacter("hitung 1111123")` -> `1`
