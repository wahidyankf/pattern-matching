## Reverse String

Diberikan suatu `string`, hasilkan sebuah `string` baru yang urutannya terbalik dari string asalnya. Contoh:

- `reverse('tulisan')` -> `'nasilut'`

- `reverse(' tulisan')` -> `'nasilut '`

- `reverse('ğŸ˜emoji')` -> `'ijomeğŸ˜'`
